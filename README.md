# 2021.04.16勉強会

## 要件

最低限の機能として
- ユーザーのCRUD
  - ユーザー一覧表示 (/users)
  - ユーザーの作成 (/users/create)
  - ユーザーの削除/編集 (/users/[userId])
- 不動産業者のCRUD
  - 不動産業者一覧表示 (/agencies)
  - 不動産業者の作成 (/agencies/create)
  - 不動産業者の削除/編集 (/agencies/[agencyId])
- 不動産のCRUD
  - 不動産一覧表示 (/properties)
  - 不動産の作成 (/properties/create)
  - 不動産の削除/編集 (/properties/[propertyId])

また、これらの画面へ遷移できるように左から開かれるドロワーメニューにリンクを表示したい
